# Approved Folder

Actions that have been reviewed and approved by human.

The orchestrator monitors this folder and executes approved actions.

## Workflow
1. File moved here from /Pending_Approval
2. Orchestrator detects new file
3. Executes the approved action
4. Logs result
5. Moves file to /Done

## Note
Files should not remain here long - they are processed automatically within minutes.
